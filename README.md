# 🛸 NEON INVADERS: OVERDRIVE EDITION

**Neon Invaders: Overdrive Edition** é um jogo de disparos arcade (Shoot 'em Up) desenvolvido em Python. O projeto combina uma estética retro-futurista vibrante com mecânicas modernas, como sistemas de partículas, árvores de melhorias (upgrades) e batalhas contra chefes desafiadores.

---

## 🚀 Funcionalidades Principais

- **Seleção de Naves:** Escolhe entre três classes distintas, cada uma com atributos de HP, velocidade e ataques especiais únicos (Beam, Shield ou Fan Shot).
- **Sistema de Progressão:** Ao final de cada onda, o jogador pode escolher upgrades estratégicos (Dano, Velocidade de Disparo, Reparação).
- **Estética Neon:** Efeitos visuais baseados em geometria neon, sistema de partículas para explosões e feedback visual de trepidação de câmara (screen shake).
- **Mecânicas de Combate:** Barreiras destrutíveis, múltiplos tipos de inimigos (Shooters e Normais) e encontros com Bosses a cada 5 ondas.
- **Persistência de Dados:** Sistema de High Score guardado localmente em JSON.

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **CustomTkinter:** Interface de utilizador moderna e estilizada.
- **Tkinter (Canvas):** Motor de renderização gráfica 2D.
- **Pygame (Mixer):** Gestão de áudio e efeitos sonoros (opcional).
- **JSON:** Armazenamento de progresso e recordes.
